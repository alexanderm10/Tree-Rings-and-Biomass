FILENAME: 

"MMS_L2_gapfilled_NEE_monthly.txt". The file contains values of aggregated NEE for each month(g C/m2/day) for Morgan Monroe State Forest (MMS). The values were calculated from AmeriFlux L2 gap filled files for MMS between 1999 and 2013 (AmeriFlux L2 files were downloaded from the ORNL server).
