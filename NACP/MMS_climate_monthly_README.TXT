FILENAME: 

"MMS_climate_monthly.txt". The file contains monthly climate variables for Morgan Monroe State Forest(MMS) between 1998 and 2012. The monthly values were calculated from hourly climate variables used to run CLM in MMS.

HEADER:

year: year (1998-2012)
month: month (1-12)
flds_mean:  mean incident longwave radiation, calculated using the mean hourly data for each month (W/m2)
flds_total: total incident longwave radiation, calculated using the sum of hourly data for each month (W/m2)
fsds_mean: mean incident solar radiation, calculated using the mean hourly data for each month (W/m2)
fsds_total: total incident solar radiation, calculated using the sum of hourly data for each month (W/m2)
prec: total precipitation in each month (mm)
psrf: mean atmospheric pressure (Pa)
rh: mean relative humidity (%)
temp: mean air temperature, calculated using the mean hourly data for each month (K)
wind: mean wind speed (m/s)
temp_max: maximum air temperature, calculated using the maximum value from hourly data for each month (K)
temp_min: minimum air temperature, calculated using the minimum value from hourly data for each month (K)
